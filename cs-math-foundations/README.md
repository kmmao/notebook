# 计算机科学的数学基础（2025版）

> 数学是真理，离散数学是计算的本质，理论计算机科学是智慧的极限

## 为什么学这个？

> "Computer Science is no more about computers than astronomy is about telescopes."  
> — Edsger Dijkstra

计算机科学的本质是数学。代码只是表象，底层是：
- **逻辑** — 程序的正确性
- **图论** — 数据结构与算法
- **组合** — 复杂度分析
- **数论** — 密码学基础
- **计算理论** — 什么能算，什么不能算

## 三条学习路线

### 路线A：算法与数据结构方向
```
离散基础 → 图论 → 组合数学 → 复杂度理论
```
**适合**：刷题、竞赛、面试准备

### 路线B：密码学与安全方向
```
离散基础 → 数论 → 代数结构 → 密码学应用
```
**适合**：安全研究、区块链、密码学工程

### 路线C：理论计算机科学方向
```
逻辑 → 计算理论 → 复杂度理论 → 形式化方法 → 类型论
```
**适合**：学术研究、编程语言设计、形式化验证

---

## 核心思想

1. **抽象** — 从具体到一般，找到本质结构
2. **证明** — 不是"我觉得对"，而是"我能证明"
3. **构造** — 存在性证明 vs 构造性证明
4. **归约** — 把新问题转化为已解决的问题
