# 第七阶段：代数结构（4-6周）

## 概述

抽象代数研究"运算的本质结构"，是密码学和编码理论的基础。

---

## 1. 群（Group）

### 定义
集合G上的运算·满足：
1. **封闭性**：a,b ∈ G ⟹ a·b ∈ G
2. **结合律**：(a·b)·c = a·(b·c)
3. **单位元**：∃e: a·e = e·a = a
4. **逆元**：∀a ∃a⁻¹: a·a⁻¹ = a⁻¹·a = e

### 阿贝尔群（交换群）
额外满足：a·b = b·a

### 常见群

| 群 | 运算 | 说明 |
|----|------|------|
| (ℤ, +) | 加法 | 整数加法群 |
| (ℤₙ, +) | 模加法 | 循环群 |
| (ℤₙ*, ×) | 模乘法 | n互素元素 |
| Sₙ | 复合 | 对称群 |
| GL(n, F) | 矩阵乘法 | 一般线性群 |

### 子群
H ⊆ G 且 H 也是群

**拉格朗日定理**：|H| 整除 |G|

### 循环群
由一个元素生成：G = ⟨g⟩ = {gⁿ : n ∈ ℤ}

---

## 2. 环（Ring）

### 定义
集合R上两个运算(+, ×)满足：
1. (R, +) 是阿贝尔群
2. (R, ×) 满足结合律
3. 分配律：a(b+c) = ab + ac

### 常见环

| 环 | 说明 |
|----|------|
| ℤ | 整数环 |
| ℤₙ | 模n剩余类环 |
| F[x] | 多项式环 |
| M_n(F) | 矩阵环 |

### 理想
I ⊆ R 满足：
- (I, +) 是子群
- ∀r ∈ R, a ∈ I: ra ∈ I

### 商环
R/I

---

## 3. 域（Field）

### 定义
(F, +, ×) 满足：
1. (F, +) 是阿贝尔群
2. (F\{0}, ×) 是阿贝尔群
3. 分配律

### 常见域

| 域 | 元素数 | 说明 |
|----|--------|------|
| ℚ | ∞ | 有理数 |
| ℝ | ∞ | 实数 |
| ℂ | ∞ | 复数 |
| ℤₚ (p素数) | p | 素域 |
| GF(pⁿ) | pⁿ | 有限域 |

### 有限域（伽罗瓦域）

对每个素数幂 q = pⁿ，存在唯一的q元域 GF(q)

**构造**：GF(pⁿ) = ℤₚ[x]/(f(x))，其中f是n次不可约多项式

**应用**：
- 纠错码（Reed-Solomon）
- 密码学（AES的GF(2⁸)）
- 椭圆曲线

---

## 4. 格（Lattice）

### 定义
```
L = {Σ aᵢbᵢ : aᵢ ∈ ℤ}
```
其中b₁, ..., bₙ是线性无关的基向量

### 基本问题

| 问题 | 描述 |
|------|------|
| SVP | 找最短非零向量 |
| CVP | 给定目标点，找最近格点 |
| SIS | 找短整数解 |
| LWE | 带噪声的线性方程组 |

### 格基约简
- LLL算法：多项式时间近似
- BKZ算法：更好的近似

### 应用
- 后量子密码学
- 整数规划
- 丢番图逼近

---

## 5. 同态与同构

### 群同态
f: G → H 满足 f(ab) = f(a)f(b)

### 同构
双射的同态

**同构意味着结构相同**

### 第一同构定理
G/ker(f) ≅ im(f)
