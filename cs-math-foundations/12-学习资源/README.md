## 学习建议

### 按兴趣选择路线

**算法竞赛方向**：
离散基础 → 组合 → 图论 → 跳过抽象部分

**密码学方向**：
数论 → 代数结构 → 椭圆曲线 → 格密码

**编程语言/形式化方向**：
逻辑 → 类型论 → 范畴论 → 证明助手

**纯理论方向**：
全部按顺序学习

### 学习方法

1. **做题**：数学需要动手练习
2. **证明**：不要满足于"理解"，要能写出证明
3. **编程**：实现算法加深理解
4. **阅读原始论文**：体验思想的诞生

### 进阶资源

- **顶级会议**：STOC, FOCS, CCC（复杂度）
- **期刊**：Journal of the ACM, SIAM Journal on Computing
- **博客**：Computational Complexity Blog, Gödel's Lost Letter
