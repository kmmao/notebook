## 1. 数学证明方法

### 直接证明
从假设直接推导出结论

```
定理：若n是偶数，则n²是偶数
证明：设n = 2k（k是整数）
      则n² = (2k)² = 4k² = 2(2k²)
      因此n²是偶数 ∎
```

### 反证法（Proof by Contradiction）
假设结论不成立，推出矛盾

```
定理：√2是无理数
证明：假设√2 = p/q（p,q互素）
      则2 = p²/q²，即p² = 2q²
      所以p²是偶数，p是偶数，设p = 2k
      则4k² = 2q²，即q² = 2k²
      所以q也是偶数
      但p,q都是偶数与互素矛盾 ∎
```

### 对偶证明（Contrapositive）
证明 P→Q 等价于证明 ¬Q→¬P

### 构造性证明
通过构造一个例子来证明存在性

### 非构造性证明
证明存在但不给出具体构造

## 2. 数学归纳法

### 普通归纳法

```
证明 P(n) 对所有 n ≥ n₀ 成立：
1. 基础步骤：证明 P(n₀) 成立
2. 归纳步骤：假设 P(k) 成立，证明 P(k+1) 成立
```

**经典例子**：
```
证明：1 + 2 + ... + n = n(n+1)/2

基础：n=1时，左边=1，右边=1×2/2=1 ✓
归纳：假设对k成立，即 1+2+...+k = k(k+1)/2
      则 1+2+...+k+(k+1) = k(k+1)/2 + (k+1)
                        = (k+1)(k/2 + 1)
                        = (k+1)(k+2)/2 ✓
```

### 强归纳法（完全归纳法）

```
归纳假设：假设 P(n₀), P(n₀+1), ..., P(k) 都成立
证明：P(k+1) 成立
```

### 结构归纳法
用于树、表达式等递归结构

## 3. 鸽巢原理

> 如果n+1个物品放入n个盒子，至少有一个盒子有≥2个物品

**应用**：
- 任意13个人中，至少2人生日在同一月
- 任意5个整数中，至少2个模4同余

### 广义鸽巢原理
> 如果n个物品放入k个盒子，至少有一个盒子有 ⌈n/k⌉ 个物品

## 4. 递归与递推

### 递归定义
```
斐波那契数列：
F(0) = 0
F(1) = 1
F(n) = F(n-1) + F(n-2), n ≥ 2
```

### 求解递推关系

**特征方程法**：
```
对于 aₙ = c₁aₙ₋₁ + c₂aₙ₋₂
特征方程：x² = c₁x + c₂
求根后得通解
```

---

## 5. 良序原理

> 自然数的任何非空子集都有最小元素

这是归纳法的基础，三者等价：
- 数学归纳法
- 强归纳法
- 良序原理
