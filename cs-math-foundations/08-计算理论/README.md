# 第八阶段：计算理论（6-8周）

## 概述

计算理论回答"什么能被计算？"——这是计算机科学最根本的问题。

---

## 1. 形式语言

### 字母表与语言

```
字母表 Σ：有限符号集
字符串：Σ上符号的有限序列
Σ*：所有字符串的集合
语言 L ⊆ Σ*
```

### 语言的运算

```
并：L₁ ∪ L₂
连接：L₁L₂ = {xy : x ∈ L₁, y ∈ L₂}
Kleene闭包：L* = {ε} ∪ L ∪ LL ∪ LLL ∪ ...
```

## 2. 乔姆斯基层次

| 类型 | 语法 | 识别机器 | 语言类 |
|------|------|----------|--------|
| 3 | 正则文法 | 有限自动机 | 正则语言 |
| 2 | 上下文无关文法 | 下推自动机 | 上下文无关语言 |
| 1 | 上下文相关文法 | 线性有界自动机 | 上下文相关语言 |
| 0 | 无限制文法 | 图灵机 | 递归可枚举语言 |

## 3. 有限自动机（FA）

### 确定性有限自动机（DFA）

```
M = (Q, Σ, δ, q₀, F)
Q：状态集
Σ：字母表
δ：Q × Σ → Q（转移函数）
q₀：初始状态
F：接受状态集
```

### 非确定性有限自动机（NFA）
δ: Q × Σ → P(Q)（可以转移到多个状态）

**定理**：DFA和NFA等价（识别相同的语言类）

### 正则表达式
与有限自动机等价

```
a         // 单个字符
ab        // 连接
a|b       // 并
a*        // 零或多个
a+        // 一或多个
a?        // 零或一个
```

### 泵引理（正则语言）
若L是正则语言，则存在p，使得任何|w| ≥ p的w ∈ L都可以写成w = xyz，其中：
1. |xy| ≤ p
2. |y| > 0
3. ∀i ≥ 0: xyⁱz ∈ L

**用途**：证明语言不是正则的

---

## 4. 下推自动机（PDA）

### 定义
在DFA基础上加入一个栈

### 上下文无关文法（CFG）

```
S → aSb | ε

生成语言：{aⁿbⁿ : n ≥ 0}
```

### 应用
- 编程语言语法
- 括号匹配
- 表达式解析

---

## 5. 图灵机（TM）

### 定义

```
M = (Q, Σ, Γ, δ, q₀, q_accept, q_reject)
Γ：带字母表（包含空白符）
δ: Q × Γ → Q × Γ × {L, R}（状态、写入、移动）
```

### 图灵机的变体
- 多带图灵机
- 非确定性图灵机
- 都与标准图灵机等价

### Church-Turing论题
> 任何"可计算"的函数都可以被图灵机计算

---

## 6. 可判定性

### 可判定语言（递归语言）
存在图灵机M，对任何输入都停机并正确回答

### 半可判定语言（递归可枚举语言）
存在图灵机M：
- 若w ∈ L，则M接受
- 若w ∉ L，则M可能拒绝或不停机

### 不可判定问题

#### 停机问题

```
HALT = {⟨M, w⟩ : M在输入w上停机}
```

**定理**：HALT是不可判定的

**证明**（对角线论证）：
假设H判定HALT，构造D：
```
D(M):
    if H(M, M) accepts:
        loop forever
    else:
        accept
```
则D(D)导致矛盾。

#### 其他不可判定问题
- Rice定理：图灵机语言的任何非平凡性质都不可判定
- Post对应问题
- 丢番图方程（希尔伯特第十问题）

---

## 7. 归约

### 映射归约
A ≤ₘ B：存在可计算函数f，x ∈ A ⟺ f(x) ∈ B

**用途**：证明不可判定性
- 若A ≤ₘ B且A不可判定，则B不可判定

---

## 8. 计算理论的意义

| 概念 | 意义 |
|------|------|
| 可判定 | 算法可以解决 |
| 不可判定 | 不存在通用算法 |
| 半可判定 | 可以验证正例 |
| 归约 | 问题间的相对难度 |

### 实践影响
- 编译器不能完美检测所有错误
- 杀毒软件不能检测所有病毒
- 程序验证有本质限制
