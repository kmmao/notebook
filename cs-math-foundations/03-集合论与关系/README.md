# 第三阶段：集合论与关系（2-3周）

## 概述

集合论是现代数学的基础，关系和函数是计算机科学的核心抽象。

---

## 1. 集合基础

### 基本概念

```
集合：无序、不重复的元素集合
∈ : 属于
∅ : 空集
⊆ : 子集
⊂ : 真子集
```

### 集合运算

| 运算 | 符号 | 定义 |
|------|------|------|
| 并集 | A∪B | {x : x∈A 或 x∈B} |
| 交集 | A∩B | {x : x∈A 且 x∈B} |
| 差集 | A-B | {x : x∈A 且 x∉B} |
| 补集 | Aᶜ | U-A（相对于全集U） |
| 幂集 | P(A) | A的所有子集的集合 |
| 笛卡尔积 | A×B | {(a,b) : a∈A, b∈B} |

### 重要恒等式

```
德摩根律：
(A∪B)ᶜ = Aᶜ∩Bᶜ
(A∩B)ᶜ = Aᶜ∪Bᶜ

分配律：
A∩(B∪C) = (A∩B)∪(A∩C)
A∪(B∩C) = (A∪B)∩(A∪C)
```

## 2. 关系（Relations）

### 定义
A到B的关系R是A×B的子集

### 关系的性质

| 性质 | 定义 | 例子 |
|------|------|------|
| 自反 | ∀x: xRx | = |
| 对称 | xRy → yRx | 婚姻关系 |
| 反对称 | xRy ∧ yRx → x=y | ≤ |
| 传递 | xRy ∧ yRz → xRz | < |

### 等价关系
满足自反、对称、传递的关系

**等价类**：[a] = {x : xRa}

**商集**：A/R = {[a] : a∈A}

**例子**：模n同余
```
a ≡ b (mod n) 当且仅当 n | (a-b)
等价类就是余数类 [0], [1], ..., [n-1]
```

### 偏序关系
满足自反、反对称、传递的关系

**Hasse图**：偏序的可视化

**例子**：
- 子集关系 ⊆
- 整除关系 |
- 小于等于 ≤

## 3. 函数（Functions）

### 定义
函数f: A→B 是满足"每个输入唯一对应一个输出"的关系

### 函数类型

| 类型 | 定义 |
|------|------|
| 单射（Injective） | f(a)=f(b) → a=b |
| 满射（Surjective） | ∀b∈B, ∃a∈A: f(a)=b |
| 双射（Bijective） | 单射且满射 |

### 函数复合
(g∘f)(x) = g(f(x))

### 逆函数
若f是双射，则f⁻¹存在且 f⁻¹∘f = id

## 4. 基数（Cardinality）

### 有限集
|A| = n，表示A有n个元素

### 可数无穷
|ℕ| = ℵ₀（阿列夫零）

**可数集**：能与ℕ建立双射的集合
- ℤ是可数的
- ℚ是可数的（对角线论证）

### 不可数无穷
|ℝ| = c（连续统）

**康托尔对角线论证**：ℝ是不可数的

```
假设[0,1]中的实数可列：
r₁ = 0.d₁₁d₁₂d₁₃...
r₂ = 0.d₂₁d₂₂d₂₃...
r₃ = 0.d₃₁d₃₂d₃₃...
...

构造 x = 0.e₁e₂e₃... 其中 eᵢ ≠ dᵢᵢ
则x不在列表中，矛盾！
```

### 基数算术
```
|A×B| = |A| × |B|
|A^B| = |A|^|B|  （B到A的函数个数）
|P(A)| = 2^|A|

2^ℵ₀ = c（实数的基数）
```
