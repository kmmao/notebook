# 英语精通计划（English Mastery）

> 目标：全英文生活、学习、工作。不是"能看懂"，而是"用英语思考"

## 为什么这是最重要的目录？

```
英语不好 → 只能看中文二手资料 → 信息延迟3-6个月 → 永远在追
英语精通 → 直接看原版 → 第一时间获取 → 领先
```

你想学的所有东西（AI、安全、CS理论），**一手资料全是英文**。

---

## 目标分级

| 等级 | 描述 | 检验标准 |
|------|------|----------|
| L1 阅读 | 看技术文档无障碍 | 读OpenAI/Anthropic博客不查词 |
| L2 听力 | 听技术播客/会议 | Lex Fridman播客听懂80% |
| L3 写作 | 写技术博客/文档 | GitHub README、PR description流畅 |
| L4 口语 | 技术讨论无障碍 | 能用英语做技术分享 |
| L5 思维 | 用英语思考 | 内心独白是英语，不再翻译 |

---

## 学习路径

### Phase 1: 输入轰炸（3-6个月）

**核心原则**：大量输入，先别急着输出

| 领域 | 行动 | 每日时间 |
|------|------|----------|
| 阅读 | 技术文档、论文、博客全切英文 | 2h+ |
| 听力 | 技术播客通勤听 | 1h |
| 视频 | YouTube技术频道（关字幕） | 1h |

**推荐资源**：
- 阅读：Hacker News、ArXiv、官方文档
- 播客：Lex Fridman、Latent Space、Practical AI
- YouTube：Andrej Karpathy、3Blue1Brown、Fireship

### Phase 2: 强制输出（6-12个月）

| 领域 | 行动 |
|------|------|
| 写作 | 所有笔记改用英文写 |
| GitHub | Commit message、PR、Issue全英文 |
| 社交 | Twitter/X技术账号用英文发 |
| 思考 | 遇到问题先用英文搜索 |

### Phase 3: 沉浸环境（12个月+）

- 手机/电脑系统语言改英文
- 内心独白刻意用英语
- 找英语技术社区参与讨论
- 考虑远程英语环境工作

## 具体执行

### 每日必做

```
早起：15分钟英文技术新闻（Hacker News）
通勤：播客/有声书
工作：遇到问题先Google英文
晚上：1篇英文技术文章精读
```

### 每周必做

- [ ] 写1篇英文技术笔记（哪怕很短）
- [ ] 看1个英文技术视频（不开字幕）
- [ ] 用英文回复1个GitHub Issue/Discussion

### 每月检验

- [ ] 能否不查词读完一篇论文Abstract？
- [ ] 能否听懂一期播客的主要内容？
- [ ] 本月写了多少英文内容？

## 心理建设

### 必须接受的事实

1. **会很痛苦** — 前3个月效率会下降50%
2. **会很慢** — 查词、理解都比中文慢
3. **会想放弃** — "看中文多快啊"的念头会反复出现

### 为什么要坚持

- 3个月后：技术阅读基本无障碍
- 6个月后：不再需要刻意切换
- 12个月后：英文成为默认语言
- 之后：**一劳永逸，所有学习效率翻倍**
