# 10-实战项目与面试

## 实战项目

### 入门项目
1. **电商网站自动化** — 用Playwright自动化一个电商网站（搜索→加购→下单）
2. **API测试套件** — 对一个开源API（如JSONPlaceholder）编写完整测试
3. **GitHub Actions集成** — 把测试跑在CI上，生成Allure报告

### 进阶项目
4. **跨平台测试框架** — 搭建支持Web+API的混合测试框架（POM + 数据驱动）
5. **移动端自动化** — 用Appium自动化一个真实App
6. **性能测试实战** — 用k6对API做负载测试，出性能报告

### 高级项目
7. **完整测试平台** — 前端（测试管理）+ 后端（执行引擎）+ 报告系统
8. **微服务测试策略** — 契约测试 + 集成测试 + E2E的组合方案

## 面试准备

### 高频面试题方向
- 测试策略设计（给一个产品，如何规划测试？）
- 自动化框架设计（POM、分层、报告、重试）
- Selenium vs Playwright vs Cypress 对比
- 如何处理Flaky Test？
- 如何提高测试执行速度？
- CI/CD中测试的最佳实践
- 性能测试指标与瓶颈分析

### 简历关键词
- 测试框架搭建（从0到1）
- 测试覆盖率提升（量化成果）
- CI/CD集成与测试左移
- 跨浏览器/跨平台测试
- 性能测试与优化
