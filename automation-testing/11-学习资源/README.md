# 11-学习资源

## 书籍

### 必读
- 《Google软件测试之道》— Google测试文化与方法论
- 《Agile Testing》— Lisa Crispin — 敏捷测试圣经
- 《xUnit Test Patterns》— Gerard Meszaros — 测试模式大全

### 进阶
- 《Continuous Testing for DevOps Professionals》— 持续测试
- 《Lessons Learned in Software Testing》— 测试经验宝典
- 《Performance Testing Guidance for Web Applications》— 性能测试

### 工具相关
- 《Selenium WebDriver实战宝典》
- 《Testing JavaScript Applications》— 前端测试
- 《Python Testing with pytest》— Brian Okken
