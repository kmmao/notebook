## 核心问题

### 市场层面
- [ ] 目标区域的汽车保有量和增长趋势？
- [ ] 周边现有加油站数量和分布？
- [ ] 新能源车渗透率对传统加油站的影响？

### 投资层面
- [ ] 一座标准加油站的总投资额？
- [ ] 投资回收期预期？
- [ ] 与"两桶油"合作 vs 民营品牌的利弊？

### 政策层面
- [ ] 当地加油站审批难度和周期？
- [ ] 环评、消防、安全等合规要求？
- [ ] 是否有区域限制或饱和政策？

### 转型层面
- [ ] 是否需要同步规划充电桩？
- [ ] 综合能源站的可行性？

## 检验标准

完成本研究后，应能回答：
1. 这个项目值不值得投？预期回报率多少？
2. 最大的风险点是什么？如何规避？
3. 需要多少启动资金？资金来源如何解决？
4. 审批流程有多长？有哪些关键卡点？
